## Error Learning Protocol

- Any failure (non-zero exit code, unexpected API error) must be logged in `ERRORS.md`.
- Format: `## [ERR-YYYYMMDD-NNN]` with Summary, Error, Fix, Metadata, CORRECT/WRONG examples, and Prevention rule.
