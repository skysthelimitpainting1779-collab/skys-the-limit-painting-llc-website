## Operating manual

Refer to the master [AGENTS.md](AGENTS.md) operating manual for guidelines, commands, and design standards.

## Commands

- **Build project**: `npm run build`
- **Lint check**: `npm run lint`
- **Run tests**: `npm test`
- **Master compile**: `powershell -ExecutionPolicy Bypass -File "..\compile-all.ps1"`
- **Project graph update**: `graphify update .`
