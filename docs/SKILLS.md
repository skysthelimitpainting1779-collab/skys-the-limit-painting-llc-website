# Team Skills

## Required

- Basic GitHub workflow: branches, commits, pull requests, and reviews.
- React component editing.
- TypeScript validation with `npm run lint`.
- Vite build workflow with `npm run build`.
- Vercel deployment awareness.

## Recommended

- SEO basics for titles, descriptions, schema, and sitemap updates.
- Accessibility basics for forms, headings, labels, contrast, and keyboard use.
- Performance basics for images, video, bundle size, and Core Web Vitals.
- Security basics for environment variables and customer lead data.

## Agent Operating Rules

- Inspect the current repo state before editing.
- Do not overwrite uncommitted human work.
- Run the smallest useful validation set before declaring completion.
- Prefer repo-specific scripts over generic tooling.
- Keep documentation aligned with the live application, not a template.
