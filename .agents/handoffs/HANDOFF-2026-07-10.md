## Lessons Learned

1. **Log ≠ learn** — Without inject + deny, agents re-fail the same class of error next session.
2. **PATH beats wrappers** — Fix Machine PATH for bash; then restart IDEs; then optional soft wrappers.
3. **Honesty contract** — Never claim full learn/heal without artifacts (`SESSION_CLOSE.md`, health, open:0).
4. **Windows agent shell** — Prefer `node scripts/...` and `.ps1` files; avoid nested `powershell -Command`.
5. **Next.js** — Never `next/dynamic` + `{ ssr: false }` in Server Components.
6. **Semgrep** — Login or explicit policy; do not silently permanent-disable as “fix.”
7. **pwsh dry-runs** — Complex JSON on the command line is unreliable; use files or node self-test.
8. **SSOT for lessons** — Prefer `.learnings/index.json` + `ERRORS_INDEX.md` + `ACTIVE_CONTEXT.md` over full ERRORS.md dumps.
